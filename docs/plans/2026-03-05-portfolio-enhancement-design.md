# HR Ops Portfolio 고도화 설계

**작성일:** 2026-03-05
**대상 레포:** hr-ops-portfolio
**배포:** Cloudflare Pages (GitHub 자동 배포)

---

## 배경 및 목표

- 1년차 HR Ops 실무자의 포트폴리오 사이트
- 1차 타깃: 대기업 HR 채용담당자
- 장기 타깃: 스타트업 / HR 테크 기업
- 경력이 쌓일수록 콘텐츠를 지속 추가할 수 있는 구조 필요
- PDF 제출 환경에서도 웹 포트폴리오를 활용할 수 있어야 함

---

## 전체 아키텍처

```
노션 DB (콘텐츠 원본)
    ↓ GitHub Actions (노션 API → content.js 자동 생성)
GitHub Repository
    ↓ Cloudflare Pages 자동 배포
포트폴리오 사이트 (index.html)
    ↓ print.html
PDF + QR코드
```

---

## ✅ Phase 1 — 채용담당자 설득 구조 재편 (완료)

### 채용담당자 여정 설계
```
3초:  히어로 — 포지셔닝 1줄 + 핵심 성과 3개 즉시 노출
30초: 성과 카드 — 문제 → 액션 → 결과 3줄 요약
3분:  케이스 스터디 — 상세 스토리 + 증거
선택: 자격증, 교육, 연락처
```

### 변경 내용

| 영역 | 현재 | 변경 후 |
|---|---|---|
| 히어로 | 긴 슬로건 + 설명 단락 | 포지셔닝 1줄 + 핵심 성과 3개 즉시 노출 |
| 성과 섹션 | Chart.js 차트 중심 | 문제→액션→결과 카드 우선, 차트는 보조 |
| DX 케이스 | 기술 나열 | "HR 문제를 시스템으로 해결" 프레이밍 |
| 신뢰 증거 | 없음 | 각 케이스에 스크린샷/문서 링크 슬롯 추가 |

### 콘텐츠 메시지 원칙
- 대기업 HR 기준: 조직 기여 + 컴플라이언스 + 안정적 실행력 강조
- DX 케이스: "기술을 썼다" → "HR 문제를 시스템으로 해결했다" 재프레이밍

---

## ✅ Phase 2 — PDF + QR 완성 (완료)

- `print.html` A4 인쇄 전용 CSS 재설계
- 인쇄 시 QR코드 자동 삽입 (qrcode.js, 웹 포트폴리오 URL)
- 회사명 입력 → 커버 문구 개인화 → PDF 저장 플로우 유지 및 개선

---

## Phase 3 — 노션 기반 콘텐츠 관리 파이프라인

### 노션 DB 구조

**케이스 스터디 DB**
- 제목 / 카테고리 (운영·채용·DX) / 상태 (발행·초안)
- 문제 / 액션 / 결과 / 성과 수치
- 증거 파일 URL / 태그

**성장 기록 DB**
- 자격증 / 교육 / 활동 / 취득일

### 자동화 파이프라인
- 트리거: 노션 DB 상태 "초안 → 발행" 변경
- GitHub Actions: 노션 API 조회 → `content.js` 자동 생성 → commit push
- Cloudflare Pages 자동 배포 → 15분 내 사이트 반영

### 기존 콘텐츠 마이그레이션
- 일회성 마이그레이션 스크립트로 `content.js` → 노션 DB 자동 이전
- 수동 재작성 불필요

### 인라인 편집 모드
- 긴급 수정용으로 유지 (노션 파이프라인의 보조 수단)

---

## Phase 4 — Cloudflare 보안 강화

- `admin.html` 경로 Cloudflare Access로 보호 (Google 계정 인증 연동)
- KV write API 토큰 만료 + rate limiting 적용
- 코드 내 시크릿/토큰 노출 감사 및 환경변수 이전
- `functions/api/` Worker 엔드포인트 보안 헤더 추가

---

## 제약 조건

- 빌드 도구 없음 — 순수 HTML/JS/Tailwind CDN 유지
- Cloudflare Pages 정적 배포 유지
- 노션 API는 서버사이드(GitHub Actions)에서만 호출 (클라이언트 노출 금지)

---

## 성공 기준

- 채용담당자가 3초 안에 "이 사람이 뭘 잘하는지" 파악 가능
- 케이스 스터디 1개 추가 시 노션에서 작성 → 15분 내 자동 반영
- PDF 출력 시 A4 레이아웃 깨짐 없음 + QR코드 포함
- admin.html 비인가 접근 차단
